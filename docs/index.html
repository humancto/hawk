Features

Everything you need to understand
your serverless architecture

🔍

Auto-Discovery

Scans 7 AWS services automatically. Point at an account, get a complete map of Lambda triggers and connections.

📊

Mermaid Export

Export to Mermaid diagrams. Paste directly into GitHub README, Notion, or any Markdown renderer.

🔄

Snapshot Diffing

Compare two scans to see what changed. Track infrastructure drift over time with deterministic JSON output.

🖥

Interactive Viewer

Bevy-powered 2D graph viewer with search, layer toggles, and node detail panels.

🔒

Security-First

Read-only API calls only. Environment variable values, secrets, and tokens are never exported.

⚡

Blazing Fast

Written in Rust. Concurrent AWS API calls with async/await. Scans complete in seconds, not minutes.

Install

Up and running in 30 seconds

Requires Rust 1.75+ and AWS credentials configured.

From crates.io

cargo install hawk-cloud

From source

git clone https://github.com/humancto/hawk.git
cd hawk && cargo install --path crates/hawk-cloud 
# Discover everything in your AWS account
hawk analyze aws all \
  --profile my-profile \
  --region us-east-1 \
  --out hawk.json \
  --pretty

# Export as Mermaid diagram
hawk export mermaid --in hawk.json --out hawk.mmd

# Compare two snapshots
hawk diff --old baseline.json --new current.json
